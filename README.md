# Flutter Base

Lightweight Flutter app scaffold to start new projects quickly.

Quick start

1. Install Flutter (https://flutter.dev/docs/get-started/install)
2. From this project folder run:

```bash
flutter pub get
flutter run
```

Files of interest

- [lib/main.dart](lib/main.dart) — entrypoint
- [lib/src/app.dart](lib/src/app.dart) — app + example home page
- [pubspec.yaml](pubspec.yaml) — dependencies and assets

Add components under `lib/src/` and assets under `assets/`.
